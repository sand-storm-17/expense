# Expense Tracker

Notion Link - https://www.notion.so/Expense-Tracker-b6020a584b15465abd09e17ab026101f

# Features

- Cards
    - All the cards you have
        - Add new card
        - Change its limit
        - Update spendings
        - Bill due date
        - Card wise insights
- Insights
    - 3 cards
        - Monthly spend
        - You owe
    - Graph for spend monthly analysis
    - TODO: Spend tracker, ref: https://dribbble.com/shots/21682576-Finance-Management-Insights
- Splitwise
    - Groups
        - Add new group
        - List all the groups
    - Activity
        - Show recent activities
        - Add new expense
        - Payback your share
    - Analysis
        - Show the monthly participation(group wise)
- Self tracker
    - Show diff cards to pay for
        - Electricity bill
        - Phone bill
        - Water bill
        - Fuel bill
        - Internet bill
        - Rent
    - Income
